1. pandas
1-1. data frame : 전체표를 의미합니다./ 데이터 프레임으로 다양한 형식의 파일로 변환할 수 있습니다.(html, excel..)
1-2. series : 칼럼을 의미합니다.
=> 데이터 프레임을 만든 이후에 제목으로 접근하면 series를 얻을 수 있습니다.
=> 제목은 첫 열의 데이터를 의미합니다.

1-3. data_frame.to_dict() : 딕셔너리 타입으로 변환할 수 있습니다.
1-4. series.to.list() : 리스트 타입으로 변환할 수 있습니다.

1-5. frame[frame.title == 'column value']
=> 해당되는 값을 찾을 수 있습니다.
=> 즉, 데이터 프레임에서 시리즈로 접근할 떄 대괄호 안에서 식을 통해 원하는 값을 찾을 수 있습니다.
=> 반환되는 형태는 데이터 프레임입니다.
=> len()의 인자로 데이터 프레임을 얻을 수 있습니다.

1-6. pandas.DataFrame(dictionary)
=> 딕셔너리 값을 데이터 프레임을 변환하는 메서드입니다.
=> ex) dic_1 = {
    "student" : ['any', 'some', 'every'],
    "scores" : [1, 2, 3]
} pandas.DataFrame(dic_1)

1-7. dataFrame.to_csv(path)
=> 데이터 프레임을 csv 파일로 변환하는 메서드입니다.

1-8. 데이터 프레임을 csv, excel.. 등 다른 파일에 저장할 떄 index를 제거하거 싶다면 index=False를 인자로 넣으면
됩니다.
