1. String split()
=> 특정 구분자를 기준으로 나누어 리스트로 변환합니다.


2. Unpacking
=> 묶여 있는 데이터를 여러 변수에 나누는 기술입니다.
=> JS 구조 분해 할당과 비슷합니다.
=> 변수 앞에 *를 사용하면 해당 변수는 리스트로 나머지를 흡수합니다.


3. map()
=> 반복 가능한 객체의 각 요소에 특정 함수를 적용하는 함수입니다.
=> map(int, ['1','2','3'])
=> [1,2,3]


4. \
=> 이를 출력하려면 \\로 처리해야 합니다.


5. r''
=> raw string
=> 문자열 내의 \ 문자가 특수 문자가 아니라 문자 그대로 출력됩니다.
