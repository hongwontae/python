1. Fies System
1-1. open()
=> 파일을 여는 파이썬에 내장된 메서드입니다.
=> open()를 호출하면 close()를 호출해 닫아야 합니다. => 메모리를 지속적으로 차지하기 때문입니다.
=> open()은 기본적으로 read 모드입니다. open('path', mode='r');
=> write와 관련된 메서드를 사용하려면 mode='w' 바꿔야 합니다./ w는 기존의 컨텐츠를 전부 삭제하고 추가합니다.
=> mode='a'(append), write => 기존의 컨텐츠에 덧붙입니다.


1-2. open()를 통해 변수에 할당하면 변수는 그에 관련된 메서드와 속성을 사용할 수 있습니다.
=> read() : 파일 컨텐츠를 읽어들입니다.

1-3. with 키워드
=> with 키워드가 파일을 직접 관리해줍니다.
=> close를 명시적으로 호출하지 않아도 작업이 끝나면 알아서 호출합니다.
with open(path) as file :
    contents = file.read()
    print(contents)
=> with으로 하나의 문을 생성하는 것이고 as로 변수에 할당합니다.
