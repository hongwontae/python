1. 문자열은 인덱스로 접근할 수 있다.
=> 프로그래밍은 이진법을 따르기 때문에 인덱스는 0부터 시작한다.
=> "Hello"[2] 

2. 문자열끼리 + 연산을 하면 접합만 한다.
print("123" + "345") => "123345"


3. Integer
=> 양수, 음수를 포함한 모든 숫자는 Integer
=> 1000 단위 계산을 위해 _ 사용할 수 있다.


4. Float
=> 소수


5. boolean
=> True/False
=> 대문자로 시작한다.

6. 다른 데이터 타입끼리 + 연산을 하면 에러가 발생한다. => 타입 에러

** 기본 데이터 타입은 4가지가 존재한다. string(인덱스를 통해 나타낼 수 있다.), integer, float, boolean
