1. inheritance
=> 기존 클래스에서 속성과 메서드를 상속받는 과정을 클래스 상속이라고 합니다.


2. 구조
=> class Fish(Animal) :
      def __init__ (self) :
          super().__init__()
=> 클래스 이름 옆에 중괄호 안에 상속받고 싶은 클래스를 가져옵니다.
=> 그리고 부모 클래스의 속성과 메서드를 얻기 위해서 super().__init__을 __init__ 함수 안에서 호출합니다.
=> super()는 상위 클래스를 의미합니다.
=> 호출하지 않으면 부모 클래스의 속성과 메서드를 사용할 수 없습니다.


3. 기능
3-1. 메서드 오버라이딩
=> 상속받은 메서드의 기능을 유지하고 새로운 기능을 넣고 싶을 떄, 상속 받은 함수를 정의하고 내부에서 
super().상속메서드()를 호출합니다. 그리고 다음 라인에서 원하는 로직을 작성하면 됩니다.
