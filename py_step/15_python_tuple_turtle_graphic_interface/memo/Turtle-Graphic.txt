1. from trutle import Turtle, Screen;
1-1. screen = Screen(); => 스크린에 관련된 클래스입니다.
1-2. screen.exitonclick() => turtle에서 정의한 그림이 스크린에 나타납니다. 클릭을 하면 창이 닫칩니다.
1-3. timmy_the_turtle.shape('turtle') => 모양
1-4. timmy_the_turtle.color('#bbb333') => 색깔 (튜플안에 r,g,b 데이터 타입도 허용합니다.)
1-5. timmy_the_turtle.forward(100) => 전진
1-6. timmy_the_turtle.right(90) => 물체 오른쪽으로 90 돌리기
1-7. timmy_the_turtle.pensize(num) => 선 굵기
1-8. timmy_the_turtle.setheading(num) => 방향을 설정합니다.
