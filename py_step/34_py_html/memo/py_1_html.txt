1. HTML이란? 
=> HyperText Markup Language
=> 웹사이트의 레이아웃과 구조를 지정하는 마크업 언어입니다.
=> 주석 <!-- -->


2. <h1> ~ <h6>
=> 1>2>3...>6
=> 글의 크기입니다. 주로 제목이나 부제를 의미할 떄 사용합니다.

3. <br/>
=> 줄바꿈 태그입니다.

4. <hr>
=> 줄 태그입니다.

5. <center></center>
=> center 태그에 포함된 태그들의 중앙 맞춤을 해주는 태그입니다.


6. html 속성
=> 브라우저에게 추가 정보를 주어 해당 HTML 요소에 변경 사항을 지정해줍니다.
=> <hr size=3/> : 줄의 두께가 커집니다.
=> <hr noshade/> : 줄의 음영이 사라집니다.
