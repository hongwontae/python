1. HTML 보일러플레이트(HTML 공동 템플릿, HTML 상용구를 의미합니다.)

1-1. <!DOCTYPE html>
=> 어떤 HTML 버전을 사용하고 있는지 알려줍니다.

1-2. <html lang="en">
=> html 태그를 사용하여 html의 시작과 끝을 알립니다.
=> html lang 속성을 통해서 기본 언어가 무엇인지 정의합니다.
=> SEO와 접근성에 도움을 줍니다.

1-3. <head>
=> 웹 페이지의 정보를 추가할 수 있습니다.
1-4. <title>
=> head 내에서 사용하고 문서의 제목을 결정합니다.
1-5. <meta>
=> html 문서에서 메타데이터를 제공하는데 사용합니다.
1-6. charset=utf-8
=> 문자 인코딩 방식을 지정하는 속성입니다.
=> 웹 페이지에서 여러 언어를 동시에 표현할 수 있습니다.
=> 문자 꺠짐 현상을 방지합니다.
=> 최신 브라우저와 웹 표준은 utf-8을 권장합니다.
