1. HTML Table

1-1. <table>
=> 전체 테이블 구조를 정의하는 태그입니다.
=> tr, td, thead, tbody, tfoot 요소를 포함할 수 있습니다.

1-2. <tr>
=> 테이블의 행을 정의합니다.
=> td, th 태그를 사용해서 하나의 가로 줄을 구성합니다.

1-3. <td>
=> 테이블의 열 데이터를 정의합니다.
=> tr 내부에 위치하며, 하나의 셀을 나타냅니다.

1-4. <thead>
=> 테이블의 머리글을 정의합니다.
=> th를 포함하여 열 제목 역할을 합니다.

1-5. <th>
=> 헤더 셀을 정의합니다.
=> 텍스트 굵게 처리, 가운데 정렬합니다.

1-6. <tbody>
=> 테이블의 본문을 정의합니다.

1-7.<tfoot>
=> 테이블의 푸터를 정의합니다.
