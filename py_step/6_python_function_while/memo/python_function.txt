1. 함수
1-1. def 키워드를 통해 함수를 선언할 수 있습니다.
1-2. def 함수이름() : => 이 순서대로 선언할 수 있습니다.
1-3. 이 후 밑줄에 동일한 들여쓰기는 해당 함수로 판단합니다.


2. 함수 호출과 for 문
2-1. for문 내부에서 함수를 호출하여 여러 번 호출할 수 있습니다.

=> for number in range(1, 7) :
    func_item(number)
=> func_item으로 정의한 함수가 6번 호출된다.


3. 함수 내부에서 전역 변수에 접근하려면 global 키워드로 접근해야 합니다.
=> global 키워드로 정의한 뒤에 밑에 줄에서 사용해야 합니다.
