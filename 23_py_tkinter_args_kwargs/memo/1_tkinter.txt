1. tkinter
=> 파이썬에서 GUI를 만들 떄 사용하는 내장된 라이브러리입니다.

1-1. window = tkinter.Tk()를 통해 화면을 생성할 수 있습니다.
=> window.mainloop()를 통해 화면을 계속 띄어놓을 수 있습니다.
1-2. window.title('') => 제목
1-3. windwo.minsize(widht=num, height=num) => 화면의 넓이 높이를 설정할 수 있습니다.

1-4. my_label = tkinter.Label() => 라벨을 생성할 수 있습니다. 튜플을 통해 설정을 받습니다.
=> Label()은 클래스이고 인자를 받습니다. 그렇기에 클래스의 속성에 접근해서 값을 변경할 수 있습니다.
=> my_label['text'] = 'chage the text'
=> my_label.config(text='chage the text')

1-5. tkinter.pack() => 위치를 조정할 수 있습니다.

1-6 my_button = tkinter.Button(text='...', commange) => 버튼을 생성할 수 있습니다.
=> command는 onClick입니다.
1-7. my_button.pack() => 위치를 조정할 수 있습니다.

1-8. my_input = tkinter.Entry() => input 상자를 생성합니다.
1-9. my_input.get() => input에 입력된 문자열을 얻습니다.


** 컴포넌트를 생성하고 레이아웃 메서드를 호출하지 않으면 GUI에서 보이지 않습니다.
