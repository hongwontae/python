1. 코드 작성방식
1-1. py는 각 명령문을 한 줄씩 작성합니다. 명령이 끝나면 자동으로 줄이 끝난다고 생각합니다.

1-2. py는 중괄호 대신 들여쓰기로 코드 블록을 구분합니다.
=> 들여쓰기를 하면 내부 블록, 들여쓰기가 없다면 외부 블록으로 판단합니다.
=> if, else, elif의 경우 조건에 대한 결과는 들여쓰기, 또 다른 조건은 들여쓰기 하지 않습니다. => 그래도 같은 블록 처리합니다.

1-3. 세미콜론을 이용해서 한 줄에 여러 명령문을 작성할 수 있습니다.

*** 들여쓰기란 오른쪽으로 네 칸을 의미합니다.


2. 이름 작성법
=> 스네이크 케이스를 사용합니다.
=> 스네이크 케이스는 각 단어를 소문자로 작성하고 단어 사이에 밑줄 _을 사용합니다.


3. len()
=> list의 데이터 개수를 알 수 있습니다.
=> 길이이기 떄문에 1부터 시작입니다. 인덱스는 0부터 시작합니다.


4. iterator rule
=> for문이나 while문에서 조건을 설정할 떄 함수의 호출 값으로 조건을 동적으로 정의할 수 있습니다.


5. if, for, while 내부에서는 전역변수에 global 키워드 없이 접근할 수 있습니다.


6. 2 % 26은 2를 26으로 나누는 것입니다. 이 떄 나눌 수 없기 때문에 자동으로 나머지는 2가 됩니다.
