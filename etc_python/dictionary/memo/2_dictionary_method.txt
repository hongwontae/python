1. dict.keys()
=> 딕셔너리의 모든 키를 반환합니다.
=> 반환값은 dict_keys 객체입니다. (iterable)


2. dict.values()
=> 딕셔너리의 모든 값을 반환합니다.
=> 반환값은 dict_keys 객체입니다. (iterable)


3. dict.items()
=> 딕셔너리의 모든 키-값 쌍을 dict_items 객체로 반환합니다.
=> 만약 해당 반환값을 for in에 돌렸을 떄 요소들은 튜플입니다.


4. dict.get(key, default=None)
=> 주어진 키에 해당하는 값을 반환하며, 키가 없을 경우 기본값을 반환합니다.
=> key - 조회할 키/ default - 키가 없을 때 반환할 기본값 (기본값은 None)


5. dict.update(other)
=> 다른 딕셔너리나 키-값 쌍을 추가하거나 덮어씁니다.
=> other - 병합할 딕셔너리 또는 키-값 쌍
=> 기존 키는 값을 덮어쓰고, 새로운 키는 추가합니다.


6. dict.pop(key, default)
=> 주어진 키에 해당하는 값을 반환하고, 해당 키-값 쌍을 딕셔너리에서 삭제합니다.
=> key - 삭제할 키/ default - 키가 없을 때 반환할 기본값 (선택적)
=> 키가 없을 경우 기본값을 반환하며, 기본값이 없으면 KeyError가 발생합니다.


7. dict.popitem()
=> 딕셔너리의 마지막 키-값 쌍을 제거하고 튜플로 반환합니다.
=> 반환값: 삭제된 키-값 쌍 (튜플 형태)
=> 딕셔너리가 비어 있을 경우 KeyError가 발생합니다.


8. dict.clear()
=> 딕셔너리의 모든 항목을 삭제합니다.


9. dict.copy()
=> 딕셔너리의 얕은 복사본을 반환합니다.
=>원본 딕셔너리를 수정해도 복사본은 영향을 받지 않습니다.


10. dict.setdefault(key, default=None)
=> 주어진 키가 딕셔너리에 없으면, 해당 키를 기본값으로 설정하고 반환합니다. 키가 있으면 기존 값을 반환합니다.
=> key - 확인할 키/ default - 키가 없을 때 설정할 기본값 (기본값은 None)
