1. class 정의
1-1. class 키워드와 클래스 이름으로 선언할 수 있습니다.

1-2. 생성자 메서드 (인스턴스 초기화)
def __init__ (self, name) :
    self.name = name;
=> JS의 constructor와 비슷합니다.

1-3. 인스턴스 메서드
def func1 (self) :
    return f"hello {self.name}"

1-4. 클래스를 호출하여 변수에 할당하면 할당된 변수를 인스턴스라고 부릅니다.


2. 클래스와 인스턴스
2-1. 클래스 속성 : 모든 인스턴스가 공유하는 변수
2-2. 인스턴스 속성 : 각 인스턴스마다 고유한 값을 가지는 변수

2-3. class 내부에서 def __init__ 생성자 함수에서 정의된 변수가 아니라면 클래스 속성입니다.
2-4. 클래스 속성은 인스턴스에서 그냥 접근할 수 있고 공유되는 변수입니다.


3. 메서드 종류
3-1. 인스턴스 메서드 : self를 첫 번쨰 매개변수로 받고 인스턴스 변수를 다룰 수 있습니다.
3-2. 클래스 메서드 : @classmethod 데코레이터로 정의하며, cls를 첫 번쨰 매개변수로 받습니다.
=> 클래스 속성에 접근하거나 클래스를 조작할 떄 사용합니다.
3-3. 정적 메서드 : @staticmethod 데코레이터로 정의하며, self나 cls를 받지 않습니다.
=> 클래스나 인스턴스와 관련 없는 독립적인 동작을 정의할 떄 사용합니다.
