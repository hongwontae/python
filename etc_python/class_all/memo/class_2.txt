4. 상속
4-1. 부모 클래스의 속성과 메서드를 물려받습니다.

4-2. 메서드 오버라이딩
=> 자식 클래스에서 부모 클래스의 메서드를 재정의할 수 있습니다.

4-3. super 키워드
=> 부모 클래스의 메서드를 호출할 떄 사용합니다.
=> 즉, 자식 클래스에서도 부모 클래스를 초기화가 필요하기 떄문에 생성자 함수에서 super를 호출합니다.
super().__init__(name)
=> 메서드 오버라이딩할 떄 전체를 재정의하는게 아니라 부모 클래스의 메서드 로직에 덮붙이고 싶다면 
super().부모메서드() 호출하고 밑에 줄부터 자신의 로직을 작성하면 됩니다.
