1. append(item)
=> 리스트의 끝에 항목을 추가합니다.

2. extend(iterable)
=> 리스트에 다른 iterable 객체(예: 리스트, 튜플 등)의 항목들을 추가합니다.

3. insert(index, item)
=> 지정된 위치에 항목을 삽입합니다.
=> index - 삽입할 위치의 인덱스, item - 삽입할 항목

4. remove(item)
=> 리스트에서 첫 번째로 일치하는 항목을 제거합니다. 

5. pop(index)
=> 지정된 인덱스 위치의 항목을 삭제하고, 그 항목을 반환합니다.
=> 인덱스를 지정하지 않으면 마지막 항목을 삭제합니다.

6. clear()
=> 리스트의 모든 항목을 삭제합니다.

7. index(item)
=> 리스트에서 주어진 항목의 첫 번째 인덱스를 반환합니다.

8. count(item)
=> 리스트에서 주어진 항목이 몇 번 등장하는지 셉니다.

9. sort()
=> 리스트의 항목들을 오름차순으로 정렬합니다.
=> reverse=True를 사용하면 내림차순으로 정렬할 수 있습니다.

10. reverse()
=> 리스트의 항목들을 순서를 반대로 뒤집습니다.

11. copy()
=> 리스트의 얕은 복사본을 반환합니다.
=> 내부 객체가 변경되면 원본과 복사본 모두 영향을 받을 수 있습니다.

12. join(para)
=> 리스트의 문자열 요소를 지정된 구분자로 연결하여 하나의 문자열로 반환합니다.
=> 매개변수: separator - 항목들 사이에 넣을 구분자
