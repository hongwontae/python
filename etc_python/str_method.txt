1. count()
=> 인자에 해당하는 문자가 문자열에 얼마나 존재하는지 int로 반환하는 메서드입니다.
=> text.count('t')

2. str.title()
=> 문자열을 제목으로 만들어줍니다. 즉, 맨 앞 문자 대문자 그외 소문자

3. str.upper()/ str.lower()
=> 대문자나 소문자로 만듭니다.

4. str.capitalize()
=> 첫 문자를 대문자로 만듭니다.

5. str.strip()
=> 선후 공백을 제거합니다.

6. str.find(para)
=> 찾고 싶은 문자열의 인덱스를 반환합니다. 존재하지 않으면 -1

7. str.replace(a,b)
=> 문자열에서 a를 b로 바꿔줍니다.
