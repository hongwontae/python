1. py package plus
1-1. 다른 이들이 정의한 파이썬 모듈에 접근하고 싶다면 python package index에서 찾을 수 있습니다.


2. 명령어
2-1. python -m pip install -U 
=> python -m : 현재 실행환경에서 pip 모듈을 호출합니다.
=> -U : 최신 버전으로 업데이트합니다.

2-2. pip install prettytable
=> 현재 path에서 설정된 pip를 사용합니다.

※ 가상 환경을 사용하는 경우 python -m pip install -U prettytable를 사용하는 것이 안전합니다.
