1. time module
1-1. time.time() => 현재시간을 초 단위로 반환합니다. float 타입입니다.
1-2. time.sleep(seconds) => 지정된 초동안 실행을 멈춥니다.
1-3. time.localtime(secs) => 초 단위 시간을 현지 시간 구조체로 변환합니다.
1-4. time.gmtime(secs) => 초 단위 시간을 UTC 시간 구조체로 변환합니다.
1-5. time.strftime(format, t) => 시간 구조체를 지정된 형식의 문자열로 변환합니다.
=> format에는 날짜와 시간을 나타내는 서식을 설정할 수 있습니다.
=> %Y(4자리 년도), %m(2자리 월), %d(2자리 일), %H(24시간제 시간), %M(분), %S(초)
