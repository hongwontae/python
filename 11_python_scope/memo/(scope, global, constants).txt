1. scope
1-1. 지역 Scope : 함수 내부에서 선언한 변수를 의미합니다.
(선언한 함수 내부에서만 유효하고, 함수 내부 변수를 우선시합니다.)

1-2. Global Scope : 전역에서 선언한 변수를 의미합니다.
(어디에서나 유효하지만 지역 scope보다 우선순위가 낮습니다.)

1-3. 파이썬에는 블록 스코프라는 개념이 없습니다.
(if문 내부에서 선언된 변수는 함수 내부에서 사용한게 아니라면 전역 변수입니다.)


2. global
=> 함수 내부에서 global이라는 키워드를 사용해서 전역 변수를 참조할 수 있습니다.
=> global이라는 키워드가 없다면 지역 범위 내에서 전역 변수를 수정할 수 없습니다.


3. constants
=> 변수 선언을 모두 대문자로 하면 해당 변수는 상수입니다.


※※ 함수 내부에서 global 키워드 없이 전역 변수를 사용하는 것은 할당만 가능하다.
