1. 파이썬 리스트 (JS의 배열과 비슷합니다.)
1-1. variable = [data, true, 'string', 20...]
=> 순서가 있는 집합된 데이터를 만들어내는 구조입니다.

1-2. 인덱스로 접근할 수 있습니다. 
=> variable[2] : 'string'
=> -를 통해 뒤에서 접근할 수 도 있습니다. 이 떄는 -1이 부터 인덱스 시작번호입니다.

1-3. variable.append() : 리스트 맨 끝에 데이터를 추가할 수 있는 메서드입니다.
1-4. variable.extend([data1, data2]) : 기존의 리스트 끝에 인자의 리스트 데이터를 추가하는 메서드입니다.
(배열 채 추가되는게 아니라 data만 추가됩니다.)
1-5. variable.choice(sequence) : 배열같은 시퀀스 중 하나를 무작위로 선택합니다.
1-6. variable.index('list_data') : 리스트 데이터를 인자에 넣으면 해당되는 인덱스를 반환합니다.


1-5. IndexError
=> 리스트 범위에 해당하지 않는 인덱스로 접근했을 떄 생기는 에러입니다.


1-6. 리스트안에 리스트를 넣어 중첩 리스트를 가질 수 있습니다.
=> JS 중첩 배열과 동일합니다.
