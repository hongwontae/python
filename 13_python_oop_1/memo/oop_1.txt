1-2. 객체지향 프로그램밍 : 작은 부분으로 나누어서 코드를 실행하는 개념 => 모듈, 클래스 단위로 코드를 실행하고 사용합니다.


2. 속성과 메서드
2-1. 속성 : 객체의 변수
2-2. 메서드 : 객체의 함수


3. 클래스를 통해 객체를 생성할 수 있고 클라스를 청사진이라고 부르기도 합니다.


4. 클래스
4-1. 클래스의 이름은 첫 문자는 대문자입니다.
4-2. 클래스를 호출하여 할당하면 객체를 전달할 수 있습니다.
4-3. timmy = turtle() : 클래스를 호출하여 객체를 전달하는 방법입니다.
4-4. 객체에서 속성과 메서드에 접근하려면 dot으로 접근합니다.


5. 불러오기
5-1. import 파일명 : 외부 파일을 사용할 수 있습니다. dot를 통해 각 Class, func, var에 접근할 수 있습니다.
5-2. from 파일명 import 변수, 클래스.. : 외부 파일에 단일 c,f,v에 접근할 수 있습니다.
=> 모듈은 외부 파일이며, 모듈에서 정의한 클래스, 함수, 변수를 dot을 통해 접근할 수 있습니다.
